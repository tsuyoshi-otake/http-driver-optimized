# @alvin0/http-driver Skills

Agent skills for [`@alvin0/http-driver`](https://github.com/alvin0/http-driver) — a fully typed per-service HTTP client for Axios and Fetch.

## Install

```bash
npx skills add alvin0/http-driver
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `http-driver` | Complete guide for using @alvin0/http-driver — service definition, driver building, retry, caching, middleware, token refresh, React/Vue integration |

## What This Skill Teaches Your Agent

- How to define services and build a driver with `DriverBuilder`
- Using `execService` (Axios) and `execServiceByFetch` (Fetch)
- Configuring retry, caching, timeout, and request deduplication
- Writing middleware (onion model)
- Setting up token refresh with request queuing
- Integrating with SWR, TanStack Query, and custom React hooks
- URL versioning strategies
- Common mistakes to avoid
